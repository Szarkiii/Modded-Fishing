# 🎣 MODDED FISHING AUTO-CLICKER

## ⚡ BUILD IN 2 STEPS (literally!)

### Windows:
1. Double-click `BUILD.bat`
2. Wait 5-15 minutes
3. Copy `build\libs\moddedfishing-1.0.0.jar` to `.minecraft\mods\`

### Mac/Linux:
1. Open Terminal in this folder
2. Run: `chmod +x BUILD.sh && ./BUILD.sh`
3. Wait 5-15 minutes  
4. Copy `build/libs/moddedfishing-1.0.0.jar` to `~/.minecraft/mods/`

## That's It!

Then:
1. Launch Minecraft with Forge
2. Start fishing minigame
3. Press **F** to auto-click
4. Watch it fish! 🎣

---

## What This Does

**Automatically clicks the correct hotbar slot** during fishing minigames where:
- Colors appear randomly (doesn't matter)
- Numbers 2-9 appear (this matters!)
- You click the matching hotbar slot

The addon detects the number and clicks automatically.

---

## Requirements

- Minecraft 1.20.1
- Minecraft Forge 47.0.0+
- Java installed
- Internet (for first build only)

---

## Troubleshooting

**"Build failed"**
- Make sure Java is installed: `java -version`
- Check internet connection
- Try again (sometimes it needs 2 attempts)

**"Java not found"**
- Download from: https://www.java.com/download/
- Restart computer after installing
- Try again

**"Stuck on downloading"**
- This is normal! First build downloads 500MB+ of dependencies
- Can take 5-15 minutes
- Let it finish
